学学Vue

Vue入门

Vue说白了就是javaScript的框架，要学好Vue要先学好javascript

html 结构；css 表现；javaScript 行为

javaScript的知名框架：

  1. jQuery说白了也是一个javaScript的框架。优点是简化了DOM操作，缺点是频繁的操作DOM。
  2. Angular：Google收购的前端框架，由一群Java程序员开发，基特点是将后台的MVC模式搬到了前端并增加了模块化开发的理念，与微软合作，采用TypeScript语法开发；对后台程序员友好，对前端程序员不太友好，缺点版本迭代不合理
  3. React：Facebook 出品，一款高性能的JS前端框架；特点是提出了新概念【虚拟DOM】于减少真实DOM操作，在内存中模拟DOM操作，有效的提升了前端渲染效率;缺点是使用复杂，需要额外学习一门【JSX】 语言

Vue的学习路线：

  • Vue的核心概念：组件、数据绑定、指令
  • 生命周期钩子、事件
  • 路由、状态管理库（Vue Router和Vuex）
  • Vue和后端服务器通信：RESTful API或WebSocket。Vue和后端沟通的相关技术和协议：AJAX、Fetch、Axios、GraphQL

1. Vue

1.1. 组件

类似于方法。一个应用中的很多功能都是组件编写的，那么这个应用就是组件化应用。

使用方法：创建组件、注册组件、使用组件

对象式写法、函数式写法

例子：

1、Vue.extend()：

  • 调用Vue.extend()创建的是一个组件构造器。
  • 通常在创建组件构造器时，传入template代表我们自定义组件的模板。
  • 该模板就是在使用到组件的地方，要显示的HTML代码。
  • 事实上，这种写法在Vue2.x文档中基本看不到了，它会直接使用下面语法糖

2、Vue.component():

  • 调用Vue.component()是将刚才的组件构造器注册为一个组件，并且给它起一个组件的标签名称。
  • 因此需要传入两个参数：a、注册组件的标签名，b、组件构造器

3、组件必须挂载在某个Vue实例下，否则不会生效

区别：

  1. 全局注册

    • 我们可以使用 Vue 应用实例的 app.component() 方法，让组件在当前 Vue 应用中全局可用。

  2. 局部注册

    • 局部注册也要使用app.component()方法
    • 区别在于局部注册要省略vue.extend直接使用Vue.component(‘组件名称’,{})

    1.2. 数据绑定

vue中有2种数据绑定的方式：
1.单向绑定(v-bind): 数据只能从data流向页面
2.双向绑定（v-model）：数据不仅能从data流向页面，还可以从页面流向data

注意事项：

1.双向绑定一般用于表单元素
2.v-model:value 可以简写为 v-model，因为v-model默认收集的就是value值

1.3. 指令

在vue中，指令是带有“v-”前缀的特殊特性（属性），指令特性的值预期是单个JavaScript表达式，语法为“v-指令:参数=”表达式””。Vue作用于HTML元素，指令提供了一些特殊的特性，将指令绑定在元素上时，指令会为绑定的目标元素添加一些特殊的行为；因此可以将指令看作特殊的HTML特性。
