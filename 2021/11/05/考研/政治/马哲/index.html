2. 第二章 唯物论

2.1. 物质运动规律意识

常见错误词：直观（完全一一对应）、消极、循环往复、直观、滞后（我们是能动的）、知行合一（我们是知行统一的）

2.2. 物质：

  1. 辩证唯物主义：各种物的总和、抽象范畴，客观实在性（不依赖于感觉而存在，唯一特性）、实践、能动、规律、承认物质的客观实在性和抽象共性
  2. 古代朴素唯物主义：把物质等同于具体形态，有局限性（五行，气）
  3. 现代形而上学的唯物主义（机器唯物主义）：形而上学谓之道，形而下学谓之器，灵魂，神，超越具体物；不辩证的，不变化的，不发展的；刻舟求剑、未来是可以确定的、原子、分子、承认物质、不承认物质运动

2.3. 运动：

  1. 一切变化、物质的根本属性和存在方式是运动，绝对的，无条件的。静止是有条件的，相对的。相对静止还是“运动的”只是特殊的稳定状态。相互依赖、相互渗透、相互包含。静止是运动的衡量尺度。

2.4. 规律：

  1. 不能改造规律，不能创造规律。只能认识、掌握、把握、学习、利用规律，不能和其他东西互相影响，别的东西影响不了规律。
  2. 自然规律不需要人参与，是无意识的。历史规律需要人参与，是有意识的。

改造创造规律发挥作用的形式方式是可以的

2.5. 意识：

  1. 人才会有的
  2. 意识–人脑–机能属性，别的词都是错的。意识是客观世界的主观印象
  3. 意识在内容上是客观的。在形式上是主观的。意识是客观内容和主观形式的统一，是物质的产物，但不是物质本身。
  4. 意识不是直观的，是能动的。
  5. 夸大意识的能动作用：通常说意识能直接创造物质

2.6. 物质和运动：

  1. 运动是物质的根本属性，物质是运动的实在基础和承担者
  2. 运动是物质的运动，物质是运动的物质
  3. 物质和运动是不可分割的，只承认物质是形而上学，只承认运动叫唯心主义
  4. 物质运动和时空的关系。时间：一维性；空间：三维性；物质运动总是在一定时间和空间中进行的，没有离开物质运动的纯粹的时间和空间，也没有离开时间和空间的物质运动。
  5. 时空是客观的和物质运动同呼吸共命运。有限无限看搭配。绝对相对看参照。时空是物质运动的存在形式。客观：意识之外；意识：观念、想法

静止是运动的衡量尺度

2.7. 物质和意识：

  1. 区别 两者的范畴不同：望梅不能真正止渴
  2. 区别 地位不同（最终根源）：物质第一性，意识第二性
  3. 联系 超出范围之外：他两都很重要，意识可以反作用与物质（意识和物质具有统一性（同一性：两者之间有联系））。意识可以转化为物质（凭借客观世界这个媒介）

2.8. 规律和意识：

  1. 尊重客观规律是正确发挥主观能动性的前提（能够成为前提的一定是客观的）
  2. 尊客（尊重客观规律）和主能（主观能动性）是辩证统一的。实践是客观规律性和主观能动性的基础。
  3. 从实际出发是正确发挥主观能动性的前提（实际是客观的）
  4. 尊客、主能的同意表达：历史趋向和主体选择、合规律性和合目的性

2.9. 物质统一性：

  1. 出现物质统一性就选，出现在选项中就选（出现他的同意转述也要选）
  2. 物质、物质属性性质等等都是无限多样的（同意转述）
  3. 半截子唯物主义：在自然观上唯物，在历史观上唯心
  4. 人和自然界相互作用、相互联系，不是相互影响、相互制约（自然就和规律一样，不能改变和创造）
  5. x依赖于物质，物质决定x
  6. 客观存在就是客观存在
  7. 自在自然：纯自然；人化自然：有人工痕迹的自然
  8. 实践是使物质世界分化为自然界与人类社会的历史前提，又是使自然界与人类社会统一起来的现实基础
  9. 主体必须是人
  10. 主观唯心主义（唯意志论）：心想事成，个人感觉、个人意识，和心有关的；客观唯心主义：神、宗教、女娲补天、上帝造人、绝对精神、理、绝对
  11. 辩证法：承认运动的变化发展；形而上：不承认运动的变化发展；可知论：思维和存在具有同一性；不可知论：思维和存在不具有同一性

3. 辩证法 联系和发展

3.1. 联系：

  1. 客观性：不是主观臆想的，不是人为的，是本身固有的，不以人的意志为转移；典型的错误：事物的联系的都是人为的
  2. 普遍性：循环，是通过中介实现的；生物链
  3. 多样性：联系和发展的环节
  4. 条件性：可以支持也可以制约，是一个中性词。条件可以改变，但不能随意

3.2. 发展：

  1. 发展是前进上升的运动，发展的实质是新取代旧，是新事物的产生和旧事物的灭亡；发展的趋势是新取代旧
  2. 新旧事务不是看时间先后，是看是否符合规律
  3. 世界不是既成事物的几何体，而是过程的集合体
  4. 任何事物都要从其暂时性去考虑

世界上一切事物都处在普遍联系中

不在普遍联系中的事物是不存在的

新出现的事物不等于新事物

3.3. 联系和发展的基本环节：

  1. 内容和形式：同样的内容可以用不同的形式去表达，形式对内容具有反作用
  2. 1）本质与现象：本质和现象都是客观的（不以人的意志为转移）；现象表现本质（客观表现客观）；意识反映物质（主观反映客观）。
    2）现象一定表现本质，本质一定会被现象所表现，本质决定现象。本质深藏于事物内部，只有通过理性思维才能把握。假象也表现本质。任何现象都一定表现本质，任何现象都一定被现象所表现，他们两者之间不会有不可逾越的界限。
    3）假象也是现象，假象也是客观的。错觉是主观的。错觉不是必然由假象引起的。感性认识是有局限的
  3. 原因和结果：前后相继and相互制约。原因和结果是相互区别的。凡事预则立不预则废。
  4. 必然与偶然：机会和机遇；必然：确定不移的趋势；偶然：不确定的趋势；都是客观的，都是不可选择的。偶然和必然可以相互转化。偶然背后隐藏着必然
  5. 现实与可能

启示：选那个 要怎样怎样的选项

3.4. 联系和发展的规律：

  1. 矛盾
    1）是客观的“辩证矛盾”。不能理解为日常逻辑上的自相矛盾，而是“对子”，“对立面的统一”的相互作用（正反义词，或者成对出现的，如祸福，师生，美丑）
    2）注意：不能调和矛盾；绝对统一和完全一样是错的（包含差别的统一）；解决矛盾的形式是多种多样的
    3）矛盾是唯物辩证法的核心
    4）矛盾的普遍性：一定会有问题，不能避免
    5）矛盾的对立统一推动事物的发展
    6）a.矛盾具有普遍性：矛盾存在于一切事物发展过程的始终。（方法论：矛盾是客观的，我们要正式矛盾并解决它，因为问题和矛盾是不能回避的，一定会出现的）；
    b.矛盾具有特殊性：每一个矛盾的各个方面在发展的不同阶段各有其特点。（方法论：具体问题具体分析，这是马克思主义活的灵魂）；
    c.主要矛盾和次要矛盾：矛盾的主要方面和次要方面的辩证关系运用到实际工作中，就是要坚持“两点论”和“重点论”的统一（既要全面看待解决问题，同时也要转重点关键和主流、大势）。针对矛盾体系的，处于支配地位的叫主要矛盾。针对具体的矛盾双方的的某一个方面，叫矛盾的主要方面（事物的性质是由主要矛盾的主要方面所规定的（事物的性质是由矛盾的主要方面规定的），方法论：所以我们评价问题要抓“主流”、“大势”）。主要矛盾：核心关键中心；矛盾的主要方面：好坏大势主流（评价）；
    d.矛盾的普遍性和相对性的关系：矛盾的普遍性=共性=一般；矛盾的特殊性=个性=个别；共性是无条件的绝对的，个性是有条件的、相对的（绝对无条件是共性、相对有条件是个性）。任何事物都是共性和个性的有机统一。（方法论：我们要举一反三，要能从个别中概括出一般）
  2. 质量互变 （只能考选择）
    1）什么是度？度是区间范围，度是保持事物质的稳定性的数量界限。在度的范围内进行变化叫做量变，如果超出度的范围，事物就会发生质变。所以我们要坚持适度原则
    2）量变：渐进性、连续性的。质变：渐进连续的中断、飞跃。量变是质变的必要准备，质变是量变的必然结果，两者是相互渗透的。量变一定引起质变。
  3. 否定之否定（从未直接考过，在选项中出现也是错误选项）
    否定是好事，是促使现存事物灭亡的因素；肯定是维持现存事物存在的因素。自我否定是正确的，外在否定是错误的。扬弃是辩证否定的实质。事物发展的规律是螺旋式上升的、波浪式前进的；不是循环往复的。

推荐纪录片：人生第一次（发展是一个过程）、监视资本主义的陷阱（算法、回声室效应）

同一代表有联系

辩证法：客观辩证法、主观辩证法；不是源与流的先后关系，而是范畴不一样，不同的表现形式而已

底线思维能力：适度原则（质量互变规律的变形）。

4. 第三章 认识论

4.1. 实践：

4.1.1. 三个特征：

  1. 直接现实性：实践与认识的最大区别就是实践的直接现实性（真正去做）实践高于认识
  2. 自觉能动性：有意识、有目的的活动（强调知道、设计创造）
  3. 社会历史性：不同时代的生产水平不同（实践是会变化的）

4.1.2. 三种基本结构：

实践的主体，客体，中介是实践活动的三要素，三者有机统一构成实践的基本结构。（实践的主客体只有通过中介才能实现）

  1. 实践主体：具有一定的主体能力，从事现实社会活动的人
  2. 实践客体：实践活动所需要的对象。可以是精神性的东西，不一定是客观的。
  3. 实践中介：各种形式的工具、手段、应用等

考研政治中的主体一定是人

客观决定主观

4.1.3. 三种类型：

  1. 物质生产实践：物质生产实践是人类最基本的实践活动
  2. 社会政治实践：
  3. 科学文化实践：

虚拟实践是实践活动的派生形式，具有相对独立性。但不是基本的实践类型之一。

科技遇到根本、第一这类词，一定要搭配生产

4.1.4. 实践在认识活动中的决定作用：

  1. 实践是认识的来源：一切真知都是从直接经验发源的。但是一个人的多数只是还是来自于间接经验（来源：实践、直接经验；直接经验和间接经验不能相互转化）（就知识本源而言，没有离开直接经验的认识，离开直接经验的知识是不存在的）

    实践和直接经验做题的时候可以看做相同的

    但是他们实质上是不同的，在范畴上不同：实践是客观范畴的，是认识论领域的“物质概念”；直接经验是主观范畴。相当于“认识概念”

  2. 实践是认识发展的动力：技术上的需要；

  3. 实践是认识的目的：读万卷书，行万里路；花开堪折直须折，莫待无花空折枝；

  4. 实践是检验认识真理性的唯一标准：只有实践才能检验认识是否具有真理性（检验和标准具有指向性，指向实践）

4.2. 认识：

4.2.1. 定义：

认识的本质是主体在实践基础上对客体的能动反映

实践：认识论中的“物质”

能动：认识论中坚持了辩证法

能动的特征：反映特性、创造特性

唯物主义的反映论：承认实践是认识的基础，承认实践第一性（实际上就是承认了物质第一性）

唯心主义先验论：否认了实践决定认识，认为认识先于物质和实践经验（就是认为意识第一性）

4.2.2. 认识的两次飞跃

1）从实践到认识

实践基础上感性认识上升到理性认识

2）从认识到实践

认识回到实践，知道实践并接受检验

3）感性认识和理性认识的区别和联系：

  1. 感性认识：认识的低级阶段；直接性、具体性；可靠性、局限性；感性认识是从实践基础上得到的，实践经验多=感性认识多
  2. 理性认识：认识的高级阶段；间接性、抽象性；深刻、正确、完整；可能会脱离现实；理性认识是通过抽象思维达到的，理性认识还是认识。
  3. 感性认识和理性认识是辩证统一的；感性认识是起点，没有感性认识就没有理性认识；相互渗透
  4. 感性渗透理性就是强调理性、思考、头脑很重要
    理性渗透感性就是强调实践体验、感官、观察、听闻很重要
    渗透谁就是谁重要

非理性因素：主体的情感、意志；作用：激活、驱动、控制

认识规律：实践到认识再到实践的反复性和无限性发展的统一

辩证唯物论的全部认识论揭示了：

  1. 认识是无限发展的过程，在形式上是循环往复的，在实质上是前进上升
  2. 认识过程的反复性和无限性
  3. 主观和客观、认识和实践的具体的历史统一
  4. 人类认识运动的基本规律

5. 第四章 唯物史观

半截子唯物主义：自然观上唯物、历史观唯心主义（历史人物在特定时期可以改变历史的发展趋势）

5.1. 社会存在和社会意识：

1）社会存在：类似于物质（历史领域中的“物质”概念）

自然地理环境：永恒、必要；对社会历史发展的作用：影响、加速、延缓；物质生产方式：决定、基础、首要前提

2）社会意识：在历史领域中的意识（想法、观念）；包括：社会意识形式、社会心理、风俗习惯

社会意识形式：分为意识形态（统治阶级的思想工具、可以加阶级前缀的；其中主要包括政治法律思想）和非意识形态（不服务于特定的阶级、不能加阶级前缀的；自然科学）

3）社会存在和社会意识的关系：a决定b，b是a的反映，b对a有能动反作用；社会存在的变化决定社会意识的变化，但是不会同步变化！社会意识内各种形式之间相互影响各自具有历史继承性

商品一定是劳动产品√ 劳动产品一定是商品×

一、劳动二因素：价值、使用价值

1）使用价值：人与自然的关系，是商品的自然属性。使用价值构成了物质内容。使用价值是差别多样的

2）价值：无差别、一般（脑力体力消耗–卡路里、在质上是相同的）、社会

交换价值：比例，由价值决定

3）关系：对立统一（对于商品而言；商品既有价值又有使用价值）、不可兼得（就人而言的，只可以拥有使用价值或者价值）

使用价值是价值（交换价值）的物质承担者；价值寓于使用价值之中（有价值一定有使用价值）；没有使用价值的一定没有价值；一种物品即使有使用价值，但如果不是劳动产品也不具有价值

绝对的寓于相对的：

矛盾斗争性寓于同一性

矛盾普遍性寓于特殊性

必然性寓于偶然性

绝对真理寓于相对真理

价值寓于使用价值

二、劳动二重性：

1）具体劳动：差别，使用价值（差别、自然、物质）

商品价值的实体=价值的本质=一般无差别劳动=抽象劳动

2）抽象劳动：没差别，价值（无差别、社会、一般）

3）关系：劳动的二重性决定了商品的二因素

没有出现个别这个词，那么默认指的是社会

2）五个职能：
