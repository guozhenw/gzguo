1. java的基本知识

1.1. 数据类型

1.1.1. 基本数据类型

一、整数（byte（1）、short（2）、int（4）、long（8））

byte ：-128到127

short ：-32768到32767

int ：直接进行使用

long ：数字后面进行后缀标识L

二、小数（float（4）、double（8））

float类型的数据，值中有小数点必须加f、F后缀

float数据范围大于long数据范围

只有大的数据才会使用科学计数法表示

三、字符（char）

编码集：数字和符号对应的表

编码集是从零开始的

A 65；a 97

四、布尔（boolean）

为什么java中要有这么多的数据类型？

为了合理的利用内存空间。这么多的数据类型需要进行数据转换

1.1.2. 数据类型转换

1、大范围的数据接收小范围的数据直接转换，自动转换

2、小范围的数据接收大范围的数据，

有了数据类型就可以定义变量了！

1.2. 变量

常量：值不能改变 （数字）

变量：值可以发生改变 （x、y）

1.2.1. 定义变量

数据类型 变量名 = 值 =====》给变量第一次赋值，初始化
int a = 10;
使用变量 ，进行值修改：
a = 100;
也可以先声明然后使用，但如果只声明不赋值就进行使用，那么就会出现未被初始化异常

定义变量的时候要注意变量的命名规范！

1.2.2. 变量的命名规范

  1. 数字（不能开头）、字母、下划线、$ 都可以用来组成变量名
  2. java关键字 保留字，具有特殊的含义，不能用来表示变量名
  3. 变量名要见名知意
  4. 驼峰命名

有了变量和类型的知识，就可以进行运算了，要运算就还需要运算符！

1.3. 运算符

1.3.1. 结果是数字的运算符

一、结果是数字的运算符又分为以下两种类型：

算术运算符：+ - * / ++ –

赋值运算符：+= *= /= %= -=

二、运算结果的类型：

  1. 如果参与运算的数据类型相同，结果默认是当前数据类型：int long float double
  2. 如果参与运算的数据类型相同，如果是byte short，那么运算结果默认是 int 类型
  3. 如果参与运算的数据类型不相同，结果默认是范围比较大的数据类型：int long float double
  4. 如果参与运算的数据类型不相同，如果是byte short，那么运算结果默认是 int 类型

三、*++ –*

  1. 变量++：先使用变量后++
  2. ++变量：先++后使用变量

四、优先级

​ 一个表达式中出现了多个运算符，就会出现优先级的问题

​ 优先级：++ > 算数优先级 > 赋值运算符

五、字符char运算结果的类型

  1. 字符和数字直接进行计算，结果的类型可以是char可以使int，结果的类型取决于接收的类型
  2. 字符和int变量进行运算，结果默认是int类型

1.3.2. 结果是关系(boolean)的运算符

结果是关系(boolean)的运算符又分为以下两种类型：

关系运算符：> >= < <= == !=

逻辑运算符：&& || ！

注意&&和||的短路运算：前面的表达式就可以决定整个表达式的结果，后面的表达式不执行

1.3.3. 按照参与运算的个数分类的运算符

  1. 一元运算符：参与运算的数据有一个
  2. 二元运算符：参与运算的数据有一个
  3. 三元运算符：参与运算的数据有一个

结果的类型与参与运算的数据类型有关

1.5.2. 类类型

一、类类型的定义：

类类型 class 名字

类就是具有相同特征和行为的一类事物

二、类名和文件名的关系：

  1. java文件中，可以包含多个类，类名默认是文件名
  2. 一个java文件中有多个类，文件名要和public类名一致
  3. java中可以有多个类，但是只能有一个是public
  4. 文件中的类都不是public的，那么java的文件名可以任意

三、属性私有化和get、set方法：

  1. get方法 获取到属性的值，可以进行其他的操作，无参数，有返回值
  2. set方法 传入数据，将数据给属性进行赋值，有参数，无返回值，也叫访问器方法
  3. 方法的形参，当前方法生效是局部变量。属性在当前类中生效是全局变量。如果重名的话 this 代表的是当前对象即全局变量

四、构造方法：

  1. new + 构造方法 =>构造方法
  2. java中只要创建了类机会默认有一个无参数的构造方法
  3. 书写格式：public 类名(){}
  4. 只要在类中进行了构造方法的书写，那么就会覆盖默认无参数的构造方法
  5. 方法名相同，参数列表不同（个数不同、顺序不同、类型不同），这种现象叫做方法重载
  6. this() 可以进行构造方法的调用，必须放在方法体的第一句

五、类和类之间的情况

  1. 两个类的属性和方法重复，但是属性只能在类中，get与set也只能在类中。我们现在就可以把两个类相同的部分抽离到一个类中！（重复的代码放在一个类中）
  2. 之前的两个类怎么才能使用刚刚创建的类的属性和方法呢？类的继承extends（子类 extends 父类）。子类继承了父类那么子类就拥有了父类的特征（单根继承）
  3. 创建了类就是在继承，如果没有通过extends指定的话，默认的父类就是Object（顶级父类）
  4. super可以向上调用，调用父类。只能在构造方法第一句，不能和 this 同时出现在一个构造方法中
  5. 表示数据范围大小的关系：父类的数据范围较大，子类的数据范围较大。小转大直接转换，大转小强制类型转换。变量可以调用什么方法，要看变量的类型。如果子类向上扩展（向上转型），那么会出现子类的扩展功能隐藏。动态绑定：当我们右键运行时，虚拟机会找到真正的new的对象的实际类型，执行实际类型中的方法
  6. 方法重写，子类要实现自己特殊的功能，而不是使用父类通用的功能
