1.2java语言基础

类型转换、scanner、控制语句、语句块的相关知识。

1. java的基本知识

1.1. 常量

值不能改变 （数字）

1.2. 变量

值可以发生改变 （x、y）

1.3. 数据类型

1.3.1. 基本数据类型

1、整数（byte（1）、short（2）、int（4）、long（8））

byte ：-128到127

short ：-32768到32767

int ：直接进行使用

long ：数字后面进行后缀标识L

2、小数（float（4）、double（8））

float类型的数据，值中有小数点必须加f、F后缀

float数据范围大于long数据范围

只有大的数据才会使用科学计数法表示

3、字符（char）

编码集：数字和符号对应的表

编码集是从零开始的

A 65；a 97

4、布尔（boolean）

为什么java中要有这么多的数据类型？

为了合理的利用内存空间。这么多的数据类型需要进行数据转换

1.3.2. 数据类型转换

1、大范围的数据接收小范围的数据直接转换，自动转换

2、小范围的数据接收大范围的数据，

有了数据类型就可以定义变量了！

1.4. 变量

1.4.1. 定义变量

数据类型 变量名 = 值 =====》给变量第一次赋值，初始化
int a = 10;
使用变量 ，进行值修改：
a = 100;
也可以先声明然后使用，但如果只声明不赋值就进行使用，那么就会出现未被初始化异常

定义变量的时候要注意变量的命名规范！

1.4.2. 变量的命名规范

  1. 数字（不能开头）、字母、下划线、$ 都可以用来组成变量名
  2. java关键字 保留字，具有特殊的含义，不能用来表示变量名
  3. 变量名要见名知意
  4. 驼峰命名

有了变量和类型的知识，就可以进行运算了，要运算就还需要运算符！

1.5. 运算

1.5.1. 运算符

1、结果是数字的运算符：

算术运算符：+ - * / ++ –

赋值运算符：+= *= /= %= -=

结果是关系的运算符

注意事项1：

  1. 如果参与运算的数据类型相同，结果默认是当前数据类型：int long float double
  2. 如果参与运算的数据类型相同，如果是byte short，那么运算结果默认是 int 类型
  3. 如果参与运算的数据类型不相同，结果默认是范围比较大的数据类型：int long float double
  4. 如果参与运算的数据类型不相同，如果是byte short，那么运算结果默认是 int 类型

注意事项2：

  1. 变量++：先使用变量后++
  2. ++变量：先++后使用变量

注意事项3：

​ 一个表达式中出现了多个运算符，就会出现优先级的问题

​ 优先级：++ > 算数优先级 > 赋值运算符
