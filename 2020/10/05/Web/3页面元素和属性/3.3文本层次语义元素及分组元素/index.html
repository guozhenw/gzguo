3.3文本层次语义元素及分组元素

文本层次语义元素及分组元素。

1. 文本层次语义元素

名称 说明
em 把文本定义为强调内容
strong 把文本定义为语气更强的强调内容
dfn 定义一个定义项目
code 定义计算机代码文本
samp 定义样本文本
kbd 定义键盘文本
var 定义变量
cite 定义引用

1.1. cite元素

创建一个引用标记，用来指明某内容的引用或参考源

1.2. mark元素

定义带有记号的文本，可以高亮显示文档中的文字以达到醒目的效果

1.3. time元素

定义日期或者时间，可以使用机器能够读懂的方式对日期和时间进行编码

2. 分组元素

元素名称 说明
div 没有任何语义的通用元素，与span对应
blockquote 表示摘自另一个源的大段内容
pre 表示其格式应被保留的内容
ul、ol 表示无序列表，有序列表
li 用于ul，ol元素中的列表项
dl、dt、dd 表示含有一系列术语和定义说明的列表
figure 表示图片
figcaption 表示figure大元素的标题

2.1. ul元素

表示不排序的项目列表，相当于word中的项目符号。无序列表使用一对标记ul标识，每一个列表项使用li

2.2. ol元素

排序的项目列表，相当于word中的自动编号功能，适用语法和ul基本相同，列表项使用li

2.3. dl元素

dl用于自定义列表，一般与dt和dd一起使用，dt定义列表名称，dd定义内容
