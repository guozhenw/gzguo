1.1Web前端开发概述

简单介绍了html、css、javascript。

1. 学习内容

1.1. Html标记

Hyper Text Markup 超文本标记语言。主要是通过HTML标记对网页中的文本、图片、声音等内容进行描述。最新版本是HTML5

1.2. css样式

Cascading Style Sheets 层叠样式表。是一种用来表现HTML或者XML文件样式的计算机语言。CSS提供了丰富的样式，控制格式的、控制布局的、控制动态效果的等。最新版是2010年推出的CSS3。CSS3通过模块化结构使设计者可以及时的调整模块的内容，方便版本的更新与发布，同时也增添了很多的新功能

1.3. JavaScript的内置函数

1995年最初由Netscape（网景公司）的BrendanEich（瑞登·艾克）设计，后来Netscape为了营销考虑，与Sun微系统达成协议，将其改名为javascript。javascript是一种基于对象和事件驱动并具有相对安全性的客户端脚本语言。运用这个脚本语言可以给HEML网页添加动态功能，响应用户的各种操作。

2. Web前端开发概述

2.1. 什么是Web？

Web是Internet上最受欢迎的一种多媒体服务系统，整个系统由Web服务器、浏览器和通信协议组成。

2.2. 什么是Web开发？

包括：浏览器端页面的制作、服务器端程序的编写

2.3. Web前端开发

Web前端开发的主要工作是把UI(User Interface用户界面)的设计图按照W3C标准做成html页面，用CSS进行布局，并且用javascript脚本语言实现页面上的前端互动

2.4. 浏览器

浏览器是指可以显示网页服务器或者文件系统HTML文件内容，并让用户与这些文件交互的一种软件

2.5. Html5

Html5是W3C组织2014年建议的标准，因此不是所有的浏览器都支持Html5，同一浏览器对Html5的支持情况也不同

2.6. Web标准

Web标准是由W3C和其他标准化组织指定的一系列标准的集合，该标准用来创建和解释基于Web的内容，网页部分的标准通过三部分来描述：结构标准、表现标准、行为标准。

2.6.1. 结构标准

用于对于网页元素进行整理和分类，相当于搭建房子的框架。

2.6.2. 表现标准

用于设置网页元素的版式、颜色、大小等外观样式，相当于房子的装修，但是房子没有任何功能。

2.6.3. 行为标准

是对网页模型的定义以及交互的编写，相当于给房子增加了功能，让房子具有功能性，使用户和房子有更好的交互。
