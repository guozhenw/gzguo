2.2HTML文档头部的标签

头部标签。

1. title

主要用于定义HTML页面的标题，即给网页取一个名字，必须位于之内

1.1. title的作用：

  • 在浏览器的标题栏中显示
  • 可以用作默认的快捷方式或者收藏夹的名称
  • 可以作为搜索引擎结果中的页面标题

1.2. 应用时应注意

  • 一个网页只能有一个标题
  • 标题的长度不超过64个字符数
  • 标题标记对之间不允许有其他的标签存在

2. meta

用于定义页面的元信息，可重复出现在头部标记中，详细信息在 HTML5语法标准中。

3. link

用link来引用外部的文件，一个页面允许使用多个link标记引用多个外部文件。

4. style

用于为HTML文档定义样式信息，位于之中。

5. script

用于在HTML页面中加载脚本文件：如javascript。

6. base

描述了基本的链接地址/链接目标，该标签作为HTML文档中所有的链接标签的默认链接。
