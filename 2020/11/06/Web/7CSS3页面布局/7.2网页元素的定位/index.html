7.2网页元素的定位

1. 网页元素的定位

CSS有三种基本的定位机制：普通流、浮动和绝对定位

  1. 普通流：相当于默认
  2. 浮动：float属性定位
  3. 绝对定位：position属性定位

1.1. 浮动

基本语法格式：选择器{float: 属性值;}

属性值：

属性值 描述
left 元素向左浮动
right 元素向右浮动
none 元素不浮动（默认）

浮动元素不再占用原文档流中的位置，所以会对其他元素排版产生影响，我们需要对元素清除浮动，基本语法格式：选择器{clear: 属性值};。它的属性值：

属性值 描述
left 不允许左侧有浮动的元素
right 不允许右侧有浮动的元素
both 清除两侧的浮动元素影响

1.2. 绝对定位

通过CSS定位设置可以实现网页元素的精确定位，定位属性包括定位模式和边偏移两部分，语法格式：选择器{position: 属性值};。属性值：

属性值 描述
static 自动定位 默认
relative 相对定位，相对于其他原文档流位置进行定位
absolute 绝对定位，相对于上一个已经确定的父元素进行定位
fixed 固定定位，相对于浏览器窗口进行定位

2. 设计导航栏

用ul和li制作导航栏

  1. 去掉项目符号可以使用text-decoration:none或者list-style-type:none
  2. 将列表项改为行排列，即内联行显示，可添加属性display:inline或者display:inline-block;
